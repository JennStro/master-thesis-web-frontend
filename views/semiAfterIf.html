If statement explanation

In Python, you write an if-statement like:

    if condition:
        method()

but there is no need for a colon (:) or semicolon (;) after the if statement in Java! If we try to write the if-statement in Java with a semicolon to make it more like Python syntax, we get: 

    if (condition);
        method();

Java reads this as

    if (condition) {} // Nothing happens even though "condition" is true!
    method();

because Java uses semicolon to end a statement, the "if (condition);" becomes just a statement that does nothing. The correct way to write the if-statement in Java is: 

    if (condition) {
        method();
    }

    // or

    if (condition)
        method();
