If statement explanation

In Python, you can write:

    if something:
        something()
        somethingElse()

and something() and somethingElse() will not be called if "something" is false. In Java, if you did the same thing:

    if (something)
        something();
        somethingElse();

Java reads this as

    if (something) {
        something();
    }
    somethingElse();

because Java does not care about the indentation, but only reads the first statement after the if-statement. Since a statement can be a block statement or a single statement, these if-statements are the same: 

    if (something) {
        something();
    }


    if (something)
        something();

but only when you have one statement! Therefore it is good practice to always use brackets when you are writing an if in Java. If you want to see a real life example of this mistake, and how scary it could be, you should check out the "goto fail" bug.
