Bitwise operator explanation

The bitwise operators (& and |) are used to operate on bits in Java (and Python), you can read more about it here.

In Python we use "and" or "or" as boolean operator:

    if boolean1 and boolean2:
        method();

in Java, this looks like:

    if (boolean1 && boolean2) {
        method();
    }

You could also use the bitwise operator for this:

    if (boolean1 & boolean2) {
        method();
    }

but it comes with a catch: even though "boolean1" is false, "boolean2" is still checked. So the code 

    A a = null;
    if (a != null & a.equals(b)) {
        method();
    }

will produce a NullPointerException, because "a.equals(b)" is checked even though a is null! To avoid this we use the boolean operator "&&" for "and" and "||" for "or".
