Integer division explanation

When we divide two integers in Python, we get the decimal result:

    print(7/5)

The code above will print the number 1.4. But this is not the case in Java! Java rounds down to the lowest integer. We try the same in Java:

    System.out.print(7/5);

but it gives an answer of 1. The "/" in Java gives the same answer as "//" does in Python. So what should we do? When we want a double (decimal) in return from Java, we also need to divide two doubles (decimals). So we need to cast the integers as doubles: 

    System.out.print((double)7/(double)5);
